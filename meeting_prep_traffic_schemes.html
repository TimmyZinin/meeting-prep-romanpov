Схема 01

Google AI Ultra

2500 видео за $125, но кредиты общие. Veo 3.1 Fast по 10 кредитов/генерацию на Ultra-плане.

Google AI Ultra
2500 клипов Veo 3.1 / мес

Семейный шеринг До 6 аккаунтов (менеджер + 5 членов). Однако кредиты НЕ умножаются, а делятся: все 25,000 кредитов распределяются между 6 пользователями из общего пула. Для 5 блогеров = ~4167 генераций на аватар при одном Ultra-аккаунте.
Гео-ограничения для России AI-функции Google заблокированы в России из-за санкций. VPN недостаточен — Google использует мультифакторную проверку (IP + регион аккаунта + цифровой отпечаток). Нужен US/EU-аккаунт + резидентные прокси.
Veo для talking head Veo 3.1 генерирует персонажей с хорошим lip-sync, но НЕ умеет клонировать конкретного человека. Подходит как генератор B-roll и креативных сцен, но не как основной инструмент talking head.

Критические риски Продавец сохраняет recovery-данные и может вернуть аккаунт. GWE-аккаунты маркируются как «non-renewable, 1-month use only». Google периодически проводит аудит и блокирует аккаунты с региональными аномалиями. Потеря данных — вопрос времени, не вероятности. Рефанды с комиссией 10–20%.
Вердикт Для тестирования возможностей Veo за $5 — допустимо. Для стабильного продакшна 5 блогеров — неприемлемый риск. Использовать только для одноразовых экспериментов с расходными аккаунтами, никогда не хранить важный контент.

Ограничения для talking head Sora не поддерживает загрузку reference-изображений с людьми (safety policy). Единственный способ создать аватар — Cameo (требует live-видео с проверкой лица). Character feature работает только для объектов и животных, не людей.
Где Sora сильна Кинематографический визуал и эмоции — близко к «Black Mirror level». Для B-roll и креативных сцен отлично. При бюджете $50–150/мес Sora Pro за $200 выходит за рамки, но ChatGPT Plus за $20/мес даёт ~30 генераций/день для дополнительного контента.
Схема 04

DaVinci Resolve + MCP

Реальная автоматизация монтажа через Python API и MCP-серверы. Studio $295 единоразово.

Автоматический монтаж через AI
LLM (Claude/GPT) → MCP-сервер → Python API DaVinci Resolve → операции. Resolve поддерживает headless-режим (-nogui).

Дополнительные функции
Что можно автоматизировать Создание проектов, импорт медиа в бины, создание таймлайнов, добавление клипов, применение LUT/грейдов, настройка и запуск рендера, управление маркерами, создание Fusion-композиций.

Требование Для внешнего скриптинга требуется Studio ($295 единоразово). Бесплатная версия позволяет скриптить только из Fusion Console. С версии 19.1 убран UIManager из бесплатной версии.

Выводы

Три ключевых инсайта

01
Один сервис не решит задачу
Veo генерирует великолепные сцены, но не клонирует конкретного человека. Sora создаёт кинематографическое видео, но блокирует загрузку лиц по safety policy. HeyGen делает идеальные talking heads, но стоит дорого при масштабировании. Оптимальная стратегия — специализированный конвейер из 3–4 инструментов.
02
HeyGen за $69 — лучший баланс
Вариант B с HeyGen Creator + Fish Audio + OpusClip + ChatGPT Plus — наиболее сбалансированный по простоте и результату. Безлимитные Avatar III видео, ~30 Sora-генераций/день для B-roll, 200 мин озвучки, автонарезка. Не требует технического стека.
03
Plati — только для тестов
Вьетнамские подписки за $5 — расходный инструмент для тестирования, не для production. А Sora Pro за $200 имеет смысл только если бюджет позволяет и нужен именно кинематографический уровень визуала для B-roll — для talking head она парадоксально слабее более дешёвых решений.
